# 智能校园AI管理系统
## 项目介绍
本项目为微信小程序端智能校园AI管理平台，依托AI技术实现校园日常事务智能化管理，面向在校师生提供便捷高效的校园服务，简化校园管理流程，提升校园生活与办公效率。

## 技术架构
- 开发框架：微信小程序原生开发
- 开发语言：WXML、WXSS、JavaScript
- 智能服务：AI智能交互、信息智能整理
- 版本控制：Git

## 核心功能
1. 校园公告资讯推送查看
2. AI智能校园咨询问答
3. 校园场地线上预约
4. 课程作息信息查询
5. 校园报修与意见反馈
6. 师生简易信息管理

## 运行方式
1. 克隆项目至本地
2. 使用微信开发者工具导入项目
3. 直接编译预览运行
